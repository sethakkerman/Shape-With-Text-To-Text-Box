# Google Slides Script: Shape With Text to Text Box

This script converts shape boxes containing text into true text boxes in Google Slides.

## Features

- Converts rectangle shapes with text to text boxes.
- Preserves the original text formatting.
- Adjusts the size and position of the new text box to account for the default text box padding.

## Installation

1. Open Google Slides and go to `Extensions > Apps Script`.

<img width="403" alt="step-01" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/e845d137-de0a-4408-9294-4c35ae4bacfe">

2. Copy and paste the code from the [code.gs](https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/blob/main/code.gs) file into the Apps Script editor.

<img width="1276" alt="step-02" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/c6dda2ca-c0ae-4d33-bae3-0f54e4e4086a">

3. Save Project by clicking the disk icon.

<img width="361" alt="step-03" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/a915df45-9497-43a2-87a8-10bfc48a4435">

4. Click the project title to rename it to whatever you want.

<img width="393" alt="step-04" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/5a613087-b73e-403e-b9c0-0899e3413dfd">
   
5. You should now see a new menu option 'Custom Scripts > Adjust Text Boxes (or whatever you named it)'.

<img width="718" alt="step-05" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/c0a435b6-ecd8-40d5-b87f-0edfe073287f">

6. The first time you use it, you will need to authorize the script to run. You will need to select your google account then click 'allow' after.

<img width="613" alt="step-06" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/e5c1b610-5893-468e-b2ed-d3530ad12bdc">

7. When you go to 'Custom Scripts > Adjust Text Boxes (or whatever you named it)' again, it will run the script.

<img width="508" alt="step-09" src="https://github.com/sethakkerman/Shape-With-Text-To-Text-Box/assets/5940990/76ca61a2-3dfb-438d-b37a-a2d96fc89e58">
